# Yelp review classifier
Приложение для определения количества звезд для отзыва Yelp

# Получившаяся модель
Ссылка на получившуюся модель: https://huggingface.co/kertys/yelp_review_classifier

# Описание
- Приложение запрашивает пользователя ввести отзыв.
- Анализирует текст отзыва с помощью модели машинного обучения (текст на английском дает более точные ответы из-за тренировочных данных).
- Выдает предполагаемую оценку количества звезд у отзыва.

# Функции
- Ввод текста
- Определение оценки текста (1 - 5 звезд)
- Вывод результата с указанием вероятности

# Инструкции
- Скачать приложение
- Установить зависимости: pip install transformers torch
- Запустить приложение через консоль с помощью команды: python classifier.py
- Ввести текст отзыва
